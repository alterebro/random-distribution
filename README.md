# random-distribution

> A `random()` distribution visualization demo built using [parcel](https://parceljs.org/) & [p5](https://p5js.org/).

![Random Distribution Visualizer](src/media/random-distribution-visualizer.gif)

#### Run (@localhost:1234)

```sh
npm start
```

#### Build

```sh
npm run build
```
